# Как развернуть VPN-сервер с маскировкой VLESS + Reality

Ниже - краткая инструкция по запуску собственного сервера.

1. Арендуйте VPS
   - Подойдет любой надежный провайдер.
   - Важно: сервер должен находиться за пределами вашей страны и, по возможности, ближе к вам по географии (для лучшей скорости).
   - Пример: ProfitServer (оплата картой, криптовалютой и рядом российских банков).

2. Создайте виртуальный сервер
   1. Выберите ОС: `Ubuntu 24.04`.
   2. Убедитесь, что есть поддержка `IPv4` (IPv6 — дополнительно).
   3. Для личного использования (1–5 устройств) обычно достаточно минимальной конфигурации.
   4. После создания сохраните данные доступа: `IP`, `root` и пароль.

3. Подключитесь к серверу по SSH
   1. Выполните команду:
      ```bash
      ssh root@<IP>
      ```
   2. Введите пароль, выданный провайдером.
   3. После входа рекомендуется сразу сменить пароль:
      ```bash
      passwd
      ```
   4. Если пароль забыли, сбросьте его в панели провайдера (раздел управления VPS/Virtual Servers).

4. Запустите скрипт установки
   - Запустите подготовленный скрипт из вашей директории с проектом.
    ```
        ./start_server.sh
    ```
   - Если ./start_server.sh не будет работать, то используй 2 скрипт 
    ```
        ./xhttp.sh
    ```

5. Получите ссылку для клиента
   - После установки появится файл `vless.txt` со ссылкой подключения.
   - Эту ссылку можно импортировать в `v2RayTun`, `Hiddify` и другие совместимые клиенты.
